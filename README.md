# Authentication System (Frontend Project)

Project Description
This is a simple Authentication System built using HTML, CSS, Bootstrap, and JavaScript.  
It includes Login, Registration, Forgot Password, and Dashboard pages.
User Registration
Login Authentication
Forgot Password (dummy)

Responsive design using Bootstrap 5
